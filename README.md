# Task Manager App

## 📱 Overview
Task Manager App displays a simple screen showing task completion status using Jetpack Compose UI components.

## ✨ Features
- Task completion message
- Image display
- Clean Compose UI layout

## 🛠️ Technologies Used
- Kotlin
- Jetpack Compose
- Android Studio

## 📚 Learning Outcome
This project demonstrates how to design **simple UI screens and align components using Compose layouts.**
