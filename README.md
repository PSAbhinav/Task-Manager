# TaskManager | Modern Student Portal

A professional, feature-rich task management dashboard designed for students. Built with React, Vite, and Firebase.

## Features
- **University Restricted Login**: Secure authentication for educational domains.
- **Dynamic Task Management**: Add, track, and manage your academic assignments.
- **Course Tracking**: Keep tabs on your enrolled subjects and their progress.
- **Academic Schedule**: An interactive timeline for your daily lectures and labs.
- **Cloud Sync**: Powered by Firebase Firestore for real-time data persistence.
- **Premium UI**: Modern glassmorphic design with support for Dark and Light modes.

## Quick Start
1. Clone the repository.
2. Install dependencies: `npm install`.
3. Start development server: `npm run dev`.

## Technology Stack
- **Frontend**: React, Vite, Vanilla CSS.
- **Backend**: Firebase Auth, Firestore.
- **Icons**: Lucide React.
